# MobileOS

Sovereign mobile operating system project by Babb Works.

**ZAKO** is a sovereign operating system standard. Every interaction it mediates is a conservation event — it has two sides, it must balance, and it must be provable. Built on AOSP, governed by Outstack, secured by Telux, encoded by BitPads and BitLedger.

**Babb Cat** (CatFlip) is the first ZAKO distribution, targeting the Cat S22 Flip for deployment in Zambia.
